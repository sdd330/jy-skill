## 简介

基于金庸武侠世界的对话式 RPG。玩家通过自然语言与智能体对话，体验武侠冒险。

**特性：**
- 自然语言交互，无需记忆指令
- 自动记忆游戏状态
- 回合制战棋战斗
- 116 个金庸角色、25 种武功、28 种物品

## 使用

```
jy              # 开始游戏 / 继续游戏
jy 帮助         # 查看帮助
```

## 代码调用

智能体通过调用 `scripts/game-engine.ts` 中的函数来执行游戏操作：

```typescript
// 创建新游戏
const state = createNewGame('主角')

// 状态查询
getStatus(state)      // 角色状态
getInventory(state)   // 背包
getSkills(state)      // 武功

// 移动
moveTo(state, '平安镇')  // 指定目的地
moveTo(state, 'random')  // 随机漫游

// NPC 交互
talkTo(state, '村长')    // 指定NPC
talkTo(state, 'random')  // 随机NPC

// 商店
buyItem(state, '铁剑')

// 物品
useItem(state, '金创药')
equipItem(state, '铁剑')

// 武功
learnSkill(state, '基本拳法')

// 休息
rest(state)

// 战斗
const battle = startBattle(state, '山贼')
attackEnemy(state, battle.enemies, 0)
useSkillInBattle(state, battle.enemies, '基本拳法', 0)
enemyAttack(state, battle.enemies)
```

## 核心公式

### 战斗伤害
```
伤害 = 武力 + 技能攻击力 - 敌方防御
左右互搏: ×1.5
武学常识加成: +武学常识/10
随机波动: ±20%
最低伤害: 1
```

### 中毒/受伤
```
中毒掉血 = 中毒值 / 10
受伤掉血 = 受伤值 / 20
最低保留 1 HP
```

### 升级
```
升级经验 = floor(100 × 1.5^(等级-1))
升级时:
  maxHp += (hpInc + random(0,3)) × 3
  attrGain = random(0, floor((资质-10)/20) + 2) + 1
  maxMp += (9 - attrGain) × 4
  武力/轻功/防御 += attrGain
```

### 武功消耗
```
内力消耗 = 基础消耗 × ((等级+1)/2)
体力消耗: 普通攻击3, 用毒2, 解毒2, 医疗4
```

## 自动记忆

游戏状态保存在 `save/game-state.json`，每轮自动更新。

## 自动开始

玩家说"开始游戏"或"jy"时：
- 无存档：自动创建角色，开始新游戏
- 有存档：直接继续

## 死亡重置

生命归零时：
1. 叙述死亡场景
2. 显示结算（等级、存活周数、资产）
3. 自动重置为新游戏
