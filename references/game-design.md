# 金庸群侠传 · 游戏设计文档

## 游戏概述

金庸群侠传是一款基于金庸武侠世界的对话式 RPG。玩家通过自然语言与智能体对话，体验武侠冒险。

## 核心玩法

### 角色系统
- 角色有 40+ 属性：生命、内力、武力、轻功、防御等
- 角色可装备武器和防具
- 角色可学习武功，武功有 10 个等级
- 角色通过战斗获得经验，升级提升属性

### 战斗系统
- 回合制战棋战斗
- 按轻功排序行动顺序
- 每回合可移动 + 攻击/使用技能/使用物品/等待
- 支持左右互搏（攻击两次）
- 中毒/受伤持续掉血

### 物品系统
- 物品分类：装备、消耗品、书籍、暗器
- 消耗品可恢复生命/内力/体力
- 装备可提升属性
- 书籍可学习武功

### 技能系统
- 技能有 10 个等级
- 内力消耗：`MpCost * ((level+1)/2)`
- 体力消耗：普通3/用毒2/解毒2/医疗4
- 技能升级：每次使用增加 1-3 经验，100 升级

### 地图系统
- 2D 网格地图
- 支持移动、NPC 交互、事件触发
- 支持随机遇敌

## 游戏公式

### 升级
```
升级经验 = floor(100 × 1.5^(等级-1))
升级时:
  maxHp += (hpInc + random(0,3)) × 3
  attrGain = random(0, floor((资质-10)/20) + 2) + 1
  maxMp += (9 - attrGain) × 4
  武力/轻功/防御 += attrGain
```

### 战斗伤害
```
伤害 = 武力 + 技能攻击力 - 敌方防御
左右互搏: ×1.5
武学常识加成: +武学常识/10
随机波动: ±20%
最低伤害: 1
```

### 中毒/受伤
```
中毒掉血 = 中毒值 / 10
受伤掉血 = 受伤值 / 20
最低保留 1 HP
```

### 武功消耗
```
内力消耗 = 基础消耗 × ((等级+1)/2)
体力消耗: 普通攻击3, 用毒2, 解毒2, 医疗4
```

### 移动力
```
移动力 = 轻功/15 + 3
```

## 配置数据

### 角色配置
- 116 个角色（金庸原著角色）
- 每个角色单独 JSON 文件
- 包含属性、技能、物品、装备

### 武功配置
- 25 种武功
- 每种武功 10 个等级
- 5 种覆盖类型：点、线、十字、矩形、菱形

### 物品配置
- 28 种物品
- 武器、防具、消耗品、经书、暗器

### 地图配置
- 4 个地图：小村、平安镇、山洞、华山、桃花岛、全真教、光明顶
- 3 个战斗：山贼袭击、山洞遭遇战、武林高手对决
- 10 个对话
